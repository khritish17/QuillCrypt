# QuillCrypt
**QuillCrypt** is a light weight portable open sourced word-processor written in Rust. 

> **Note:** The project "**QuillCrypt**" is under development, few updates can be made which may affect but not limited to - Documentation, APIs, source code, features, underlying frameworks etc., hence stay update with the repo for recent changes

### Backend
- Rust
### Framework/Libraries
- Typst (For math mode)
### Frontend
- Flutter/Qt/Slint

(preferably slint because of its light weight and fast nature)
### Features:
- **Portable**: No installation required on device
- **Math Mode**: Natively write mathematical function-equations within the document
- **Versioning System**: Ships with inbuilt versioning system
- **Secure documents**: All documents are encrypted
